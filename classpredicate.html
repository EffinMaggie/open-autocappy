◆ assume()

set predicate::assume (   value:maybe)
inline

Override value, if different.

Allows 'unsticking' predicates that may not be able to express their condition perfectly, or to allow overrides during troubleshooting - by a human, or otherwise.

Note: the predicate will only assume a value that it doesn't have, and the next time the predicate finds the assumed value matches the tested value, it reverts automatically to normal behavior, discarding the assumption.

@emits 'value-changed' if the assumed value is set.
